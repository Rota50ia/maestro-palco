## PRÓXIMOS PASSOS IMEDIATOS

### A2.5 — Montar (pipeline parcial, só montagem)
```bash
cd /root/AIOS_PARA_VPS/projeto-video

# Corrigir durações no manifest (usar ffprobe)
python3 -c "
import json, subprocess, os
BASE = 'A2.5/audio'
m = json.load(open(f'{BASE}/audio-manifest.json'))
for a in m['audios']:
    f = f\"{BASE}/{a['arquivo']}\"
    if os.path.exists(f):
        r = subprocess.run(['ffprobe','-v','quiet','-show_entries','format=duration','-of','csv=p=0',f], capture_output=True, text=True)
        a['duracao_segundos'] = round(float(r.stdout.strip()), 1)
        print(a['cena'], a['duracao_segundos'])
json.dump(m, open(f'{BASE}/audio-manifest.json','w'), indent=2)
"

node assemble-video.js A2.5

# Se concat falhar (bug conhecido), rodar direto:
cd A2.5/montagem && ffmpeg -y -f concat -safe 0 -i concat.txt -c:v libx264 -preset fast -crf 20 -c:a aac -b:a 128k ../output/A2.5-final.mp4
```

## BUG CONHECIDO — concat assemble-video.js
